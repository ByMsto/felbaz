# 🕵️ فێڵباز — یاری ئۆنڵاین

یاری کوردی "کێ فێڵبازە؟" بە شێوازی مۆدێرن و ئۆنڵاین

---

## 🚀 چۆن دەست پێ بکات

### پێشمەرج
- **Node.js** پێویستە (نسخەی 16+)
- ئینتەرنێت بۆ بارکردنی فۆنت و Socket.io CDN

### دامەزراندن

```bash
# 1. پاکێجەکان دامەزرێنە
npm install

# 2. سێرڤەر دەست پێ بکە
npm start

# یان:
node server.js
```

### گەیشتن
سێرڤەر دەکرێتە سەر: **http://localhost:3000**

---

## 🎮 چۆن یارییەکە ئەنجام دەدرێت

1. **هۆست** سەری دەخاتەوە → ناوی خۆی دەنووسێت → ژووری نوێ دروست دەکات
2. هۆست **کۆدی ژوور** یان **لینک** دەستێنێتە هاوڕێکانی
3. هاوڕێکان بە ژووری کۆد یان لینکەکەوە **دێنە ژوورەوە**
4. هۆست **کاتێگۆری** و **ژمارەی فێڵباز** دیاری دەکات
5. هۆست یارییەکە **دەست پێ دەکات**
6. هەر کەسێک **کارتی خۆی** دەبینێت (فێڵبازەکە وشەی نییە)
7. **گفتوگۆ** بکەن دەربارەی وشەکە بە نزیک نەکردنەوەی ئاشکرا
8. **دەنگ** بدەن بۆ کەسێک کە دەزانن فێڵبازە
9. **ئەنجام** ئاشکرا دەبێت!
